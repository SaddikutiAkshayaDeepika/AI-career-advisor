 # 🎯 AI Career Advisor System

An intelligent career guidance system that analyzes your skills, identifies gaps, and provides personalized learning roadmaps.

## ✨ Features

- **Skill Gap Analyzer** - Compare your skills with industry requirements
- **Internship Recommendations** - Get targeted internship suggestions
- **Learning Roadmap Generator** - Receive step-by-step learning paths
- **5 Career Paths** - Full Stack, Data Science, Mobile Dev, DevOps, AI/ML

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Responsive Design

## 📦 Installation

1. Clone the repository
2. Open `index.html` in your browser

## 🎮 Usage

1. Select your desired career goal
2. Enter your current skills (comma-separated)
3. Click "Analyze My Profile"
4. View results and recommendations
